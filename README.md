# Pages

Код рабочих страниц без сборки веб-платформы (сервис заказчик - исполнитель).
Мультиязычный ресурс для нескольких стран.

## Реальзация 

- Оплата заказа
- Редактирование заказа 

## Рабочий стек

* REACT / react-hooks
* TypeScript
* Graphql
* Apollo
* Effector
* Styled-components
* Webpack
